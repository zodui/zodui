# Descriptor

某单个元素的描述信息，在视图上去展示相应的原数据，并且负责对数据进行校验提示。

它仅需要接受一个尽可能简单的 Zod Schema，如果接收到的是一个复杂的 Schema，那么应该尽可能简单的渲染。
